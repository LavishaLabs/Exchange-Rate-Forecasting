# Exchange-Rate-Forecasting
## Overview
This project forecasts USD to Australian Dollar exchange rates using time series
forecasting methods.
## Models Used
- ARIMA
- Exponential Smoothing
## Features
- Time series visualization
- Stationarity testing
- ACF and PACF analysis
- Forecasting
- Error metric evaluation
- Model comparison
## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Statsmodels
- Scikit-learn
